General Commands

Use ~~help [command name] for the usage of commands

  • Avatar : Sends The Avatar of the user
  • intro : introduces the bot
  • totalcmds : Sends the total number of commands
  • ping : Sends the latency of the bot
  • say : says what you want it to say
  • calc : calculates an expression
  • lyrics : Sends the lyrics of a song
  • ascii : Sends Ascii form of a text
  • wiki : Searches in wiki
  • edits : Shows the last edited message
  • bots : Shows the number of bots in the server
  • humans : Shows the number of humans in the server
  • date : Sends the date
  • setbb : Sets your birthday
  • creatembed : Create your own embed
  • bye : Says bye to a person
  • remind : sets a reminder
  • reminders : Sends The reminders set
  • userinfo : Sends info about an user
  • serverinfo : Shows info about the server
  • mem : Sends The total members in a server
  • credits : Shows credits of the bot
  • botinfo : Sends info on the bot
  • lenny : Sends a lenny face
  • swd : Sends a famous star wars quote
