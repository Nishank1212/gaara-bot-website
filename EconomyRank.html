Economy Commands

Use ~~help [command name] for the usage of commands

  • balance : shows balance of a person
  • beg : begs for money
  • deposit : deposits amount in the bank
  • withdraw : withdraws money from bank
  • slots : plays a game of slots with bet amount
  • rob : robs a person
  • search : Searches for money
  • give : gives money to a person
  • buy : buys items from shop
  • fish : fishes for fished :D
  • hunt : hunts for animals
  • sell : sells items
  • inventory : shows your inventory
  • shop : Shows the shop
  • postmeme : posts meme for money
  • disable rob : disables rob command [default disabled]
  • enable rob : enables rob
  • gamble : gambles for money
    • Rank Commands
  • rank : shows your rank
  • leaderboard : shows tehe leaderboard of the server
  • levelenable : enables levelling [default disabled]
  • leveldisable : disables levelling
  • ignorechannel : ignores channel for xp
  • ignorechanneladd : adds a channel for ignoring
  • ignorechanneldelete : deletes all channels which are supposed to be ignored for xp
  • ignorechannelremove : deletes a channel from ignore channel
  • togglepoints : toggle point for giving xp per message
