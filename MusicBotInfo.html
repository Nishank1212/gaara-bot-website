General Commands

Use ~~help [command name] for the usage of commands

  • join : join the vc you are in
  • leave : leaves the vc it is in
  • play : plays a song (if you want to play your playlist , type '~~play myqueue')
  • skip : Skips the current song
  • queue : shows the queue
  • pause : pauses the song
  • resume : resumes song
  • stop : stops the song and deletes the queue
  • remove : removes song from queue
  • volume : sets volume
  • makeplaylist : make your won playlist!
  • addtoplaylist : add a song to your playlist!
  • removefromplaylist : remove a song from your playlist
  • viewplaylist : view your playlist
  • server : gives link to support server
  • github : gives link for github page
  • invite : gives invite link
  • website : gives website link
