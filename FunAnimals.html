Fun Commands

Use ~~help [command name] for the usage of commands

  • joke : Sends a joke to the channel
  • meme : Sends a meme to the channel
  • Good Joke : Sends a really really good joke :D
  • Pat : shows a pat image
  • spoonfeeding : shows something to keep in mind about spoonfeeding
  • 8ball : asks the mysterious 8ball a question :D
  • Inspire : Sends a inspirational quote
  • Gaara : Sends info about gaara from naruto
  • hehe boi : Sends a hehe boi pic
  • khopdi tod : Sends a khopdi tod pic
  • snipe : Shows the Last Deleted message
