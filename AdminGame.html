Admin Commands

Use ~~help [command name] for the usage of commands

  • kick : Kicks User out of server
  • ban : Bans user
  • warn : Warns a person [5 warns = ban]
  • case : shows the number of warns
  • cw : clears warns for a person
  • setprefix : Sets prefix for the server
  • addrole : Adds role to a person
  • unrole : removes role form a person
  • set_welcome_message : sets welcome message
  • set_welcome_channel : sets the channel for the message to send
  • slm : sets leave system
  • create : creates a channel
  • delete : deletes a channel
  • slowmode : sets slowmode
  • modon : caps limit mod
  • modoff : switches caps limit off
  • filteron : sets curse word filter
  • filteroff : deletes filter system
  • mute : mutes a person
  • unmute : unmutes a person
  • lock : locks a channel
  • unlock : unlocks a channel
    • Game Commands
  • rps : Rock paper scissors!
  • guess : Guess the number!
  • dice : plays a game of dice
